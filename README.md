# Agentic RAG using Autogen for eCommerce

An advanced Retrieval-Augmented Generation (RAG) system using AutoGen for intelligent eCommerce product recommendations and customer support.

## Features
- Product recommendation engine
- Customer query resolution
- Knowledge base integration
- Real-time inventory updates

## Tech Stack
- AutoGen
- RAG (Retrieval-Augmented Generation)
- Python
- Vector Database

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
